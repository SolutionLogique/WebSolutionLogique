Télémaintenance RustDesk

Solution de télémaintenance rapide et sécurisée recommandée

Instructions

  1. 1Téléchargez RustDesk en cliquant sur le bouton ci-dessus (EXE recommandé pour une utilisation immédiate)
  2. 2Lancez le fichier téléchargé (l'EXE ne nécessite pas d'installation)
  3. 3
    Communiquez au technicien :
    L'ID affiché dans RustDesk (série de chiffres)
    Le mot de passe temporaire affiché en dessous
  4. 4Le technicien pourra alors se connecter à votre poste pour vous assister

Connexion sécurisée et privée

RustDesk utilise un chiffrement de bout en bout pour garantir la sécurité de votre session. Vos données restent privées et la connexion est temporaire.

Connexion NinjaOne

Alternative si vous disposez déjà d'un code NinjaOne

Connexion à la télémaintenance

Instructions

  1. 1Contactez notre support pour obtenir votre code de connexion
  2. 2Entrez votre code de connexion dans le champ ci-dessus
  3. 3Cliquez sur "Se connecter" pour accéder à NinjaOne
  4. 4Suivez les instructions à l'écran pour établir la connexion
  5. 5Notre équipe pourra alors vous assister à distance en toute sécurité

Connexion sécurisée

Vous êtes redirigé directement vers notre plateforme NinjaOne sécurisée. Toutes les sessions de télémaintenance sont chiffrées de bout en bout.
